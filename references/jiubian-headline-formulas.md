# 九边pro风格爆款标题公式库

> 来源：热点刀锋技能第六步集成（2026-05-23）
> 适用场景：微头条、头条号、公众号等中文内容平台

## 九边标题三公式

### 公式1·痛点直击式

**模板：** `XX越想越累，说白了就是你没看透这件事`

**适用场景：** 社会/职场/民生类话题

**变体：**
- `XX越想越不对劲，说白了就是...`
- `XX越想越焦虑，说白了是你没搞懂这件事`

**示例：**
- `欧洲军购美国占比从51%降到34%，说白了就是跟美国划清界限`
- `房价越涨越焦虑，说白了是你没看透这件事`

---

### 公式2·反常识式

**模板：** `别再被误导了！关于XX，90%的人都想错了`

**适用场景：** 国际/商业/政策类话题

**变体：**
- `别再被XX骗了！关于这件事，90%的人都想错了`
- `关于XX，你可能一直想错了`

**示例：**
- `别再被误导了！关于欧洲独立防务，90%的人都想错了`
- `别再被XX骗了！关于AI取代工作，90%的人都想错了`

---

### 公式3·干货觉醒式

**模板：** `看透XX的底层逻辑，你会少走很多弯路`

**适用场景：** 科技/AI/认知类话题

**变体：**
- `看透XX的本质，你会少走很多弯路`
- `看懂XX的底层逻辑，你就赢了`

**示例：**
- `看透欧洲独立防务的底层逻辑，你会少走很多弯路`
- `看懂AI的底层逻辑，你会少走很多弯路`

## 标题评分标准（满分10分）

| 维度 | 评分 | 说明 |
|------|------|------|
| ≤25字 | +2分 | 标题长度控制在25字以内 |
| 有数字+反差感 | +2分 | 如"从51%降到34%"、"涨了3倍"等 |
| 无"竟然""必须""震惊"等绝对词 | +1分 | 头条算法打压绝对词 |
| 慎用问号 | +1分 | 头条算法打压问号，尽量少用 |
| 包含情绪关键词 | +1分 | 无奈/讽刺/愤怒/划清界限等 |
| 符合九边标题三公式之一 | +2分 | 必须匹配上述三公式之一 |
| 数据冲击开场 | +1分（额外） | 如"从51%降到34%"放在标题开头 |

## 全自动选择逻辑

```
对每个话题：
  1. 按三公式各生成1个标题 → 共3个候选标题
  2. 按评分标准自动打分
  3. 选最高分标题作为最终标题
  4. 定时任务模式下不等待用户确认，直接采用
```

## 使用建议

1. **优先使用公式1** — 数据+反差感最容易触发算法推荐
2. **避免绝对词** — "震惊""重磅""必须"等词会被头条算法降权
3. **数字前置** — 如"从51%降到34%"放在标题开头，额外+1分
4. **情绪词点睛** — "说白了""划清界限""各怀鬼胎"等九边招牌词增加辨识度
