# 辣度对比案例：人形机器人价格大跳水（2026-06-10）

> 用户评价：第一稿"辣度不够，观点不够清新"，第二稿"不错"。
> 根因差异：第一稿是现象描述，第二稿是观点翻转。

## 第一稿（❌ 被拒 — 标准版，~300字）

核心论点：机器人降价 → 工人可能被替代 → 跟你有关
问题：论点太浅，90%的人看到这个话题都会这么说。没有翻转。

## 第二稿（✅ 通过 — 过瘾版，599中文字）

核心论点：降价砸的不是波士顿动力的价格，砸的是人的议价权。
翻转点：
1. "你以为机器人对手是波士顿动力？不是。是中国2亿蓝领工人。"
2. "2万块，一个工厂工人半年的工资。买回去能用五年。这笔账，哪个老板算不明白？"
3. "进步是真实的。代价也是真实的。只是承担代价的人和享受进步的人，从来不是同一批。"

## 关键差异

| 维度 | 第一稿 | 第二稿 |
|------|--------|--------|
| 观点深度 | 现象描述（降价→失业） | 结构性矛盾（降价=人的议价权被砸） |
| 身份重定义 | 无 | "你以为对手是波士顿动力？是中国2亿蓝领" |
| 数据密度 | 2-3个 | 6+个（98万/5万/2万/700倍/200万/月薪5000） |
| 具体物件 | 机器人 | 机器人+iPhone+社保+日结工资+收费站+富士康 |
| 反驳成本 | 高（"机器人确实会替代人"→共识） | 低（"机器人替代不了我"→坐办公室的人必须反驳） |
| 结尾追问 | "你会买一台放家里吗？" | "你家附近有工厂吗？最近有没有裁员的消息？" |
| 冷幽默 | 无 | "不会在第11个月的时候要求涨薪否则就走人" |
