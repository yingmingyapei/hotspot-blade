# 头条号数据驱动优化工作流（2026-06-02）

> 基于头条号数据诊断，建立数据驱动的内容优化闭环。

## 核心发现

### 问题诊断
- **点击率1.2%**（远低于5%晋级线）
- **推荐量28,097**（算法给了初始推荐）
- **阅读量341**（点击率低导致）
- **互动率0.04%**（结尾追问不具体）

### 根因分析
- 热榜数据源 ≠ 头条用户兴趣
- 科技类内容点击率0%，互动率0.06%
- 发布时间不在黄金时段
- 标题缺乏数字、反差、九边式定性

## 头条适配性评分机制

### 评分标准（1-10分）
```
1. 是否有矛盾/靶子？（九边风格核心）+3分
2. 是否涉及普通人利益？+2分
3. 是否有争议空间？（50/50对撞）+2分
4. 是否有具体数字/案例？+1分
5. 是否贴近中国社会现实？+1分
6. 领域加分/减分：
   - 国际政治/外交：+1分（实测点击率10.8%-13.7%）
   - 社会民生/道德争议：+1分（实测700+展现）
   - 军事/国防：+0.5分
   - 商业/财经：0分
   - 科技/AI：-1分（实测0.06%互动率）
   - 海外信息差：-0.5分
```

### 分级处理
| 评分 | 级别 | 处理方式 |
|------|------|---------|
| 8-10分 | S级 | 优先选择，直接进入写作 |
| 6-7分 | A级 | 可选，需要强化标题钩子 |
| 4-5分 | B级 | 降级处理，仅在候选不足时使用 |
| <4分 | C级 | **直接排除**，不适配头条用户 |

## 选题权重配置（数据驱动）

| 选题类型 | 点击率实测 | 建议权重 | 选题策略 |
|---------|-----------|---------|---------|
| 国际政治/外交 | 10.8%-13.7%（最高） | 40% | 优先选择，每天至少1-2篇 |
| 社会民生/道德争议 | 5%-10% | 25% | 贴近生活，情绪共鸣强 |
| 军事/国防 | 3%-5% | 15% | 热词加持，适度选择 |
| 商业/财经 | 1%-3% | 10% | 需降维解读，强化标题钩子 |
| 科技/AI | 0%-0.14%（最低） | 5% | 慎用！必须降维 |
| 海外信息差 | 2%-5% | 5% | 信息差优势，适度选择 |

**选题硬约束**：
- 每天5篇中，国际政治+社会民生必须占3篇以上
- 科技类每天最多1篇，且必须通过"科技降维检查清单"
- 头条适配性评分<6分的话题直接排除

## 标题公式白名单（实测高点击率）

### 高点击率公式（10%+）
| 公式 | 模板 | 适用场景 |
|------|------|---------|
| 数字+反差+定性 | "PMI 49.5%，但你的体感比数据更诚实" | 宏观经济/社会现象 |
| 信仰碎裂式 | "'永不卖出'信仰碎了" | 投资/信仰/承诺 |
| 时间跨度+情感 | "31年回避，31年隐忍，终于敢..." | 国际政治/历史 |
| 价格/数字对比 | "X万到Y万：你的命..." | 社会民生/消费 |
| 人物+冲突+道德 | "XX一席话，扯下了...遮羞布" | 社会民生/道德争议 |

### 低点击率公式（避免使用）
- 纯新闻式标题（无数字、无反差）
- 科技专业术语标题
- 太长标题（>30字）
- 无具体数字的抽象标题

## 科技类内容降维检查清单

### 标题降维
- 标题是否包含专业术语？→ 必须替换为通俗表达
- 标题是否突出"对普通人的影响"？→ 必须包含"你的XX"
- 标题是否有具体数字？→ 必须有数字冲击

### 正文降维
- 第一段是否用生活化类比？
- 是否避免了技术细节？
- 是否用"你每天用的XX"来类比？

### 结尾降维
- 追问是否具体到普通人有话说？
  - ✅ "你用过AI写代码吗？体验怎么样？"
  - ❌ "你怎么看AI编程的未来？"

## 结尾追问模板库

### 高互动率追问（实测有效）
- "你家那边呢？涨了多少？"（具体数字追问）
- "你猜是A赢了，还是B赢了？"（二选一追问）
- "你用过XX吗？体验怎么样？"（体验追问）
- "你身边有这样的人吗？"（身边故事追问）
- "如果是你，你会怎么选？"（选择追问）

### 避免使用
- "你怎么看？"（太泛）
- "评论区聊聊"（无具体方向）

## 数据回流检查机制

### 自优化规则
```
规则1：如果某类选题平均点击率>10%，提升该类权重5%
规则2：如果某类选题平均点击率<3%，降低该类权重5%
规则3：如果科技类平均点击率<1%，暂停科技类7天
规则4：如果某标题公式连续2篇点击率<1%，标记为"低效公式"
规则5：如果某标题公式点击率>10%，标记为"高效公式"
```

### 每周数据回流检查
- 已获取过去7天所有微头条的展现/阅读/评论/点赞数据
- 已计算每篇的点击率和互动率
- 已标记高效/低效标题公式
- 已更新选题权重配置
- 已检查科技类内容表现（是否需要暂停）
- 已记录本周最佳实践（哪篇爆了、为什么爆）
