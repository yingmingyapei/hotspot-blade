# 文章改写去敏感化 + 图片提示词生成

> 来源：2026-06-17 实战验证——用热点刀锋风格改写现有文章，去除敏感词违禁词

## 触发条件

用户发送已写好的文章（通常来自公众号/头条/自媒体），要求：
- "用热点刀锋风格改写"
- "去除敏感词违禁词"
- "改写一下这篇文章"

**与标准热点刀锋的区别**：不是从热榜选题生成新内容，而是对现有文章做风格改写和去敏感化。

## 改写流程

### 第一步：识别敏感词和违禁词

**常见敏感词类别**：

| 类别 | 原文用词 | 改写策略 |
|------|---------|---------|
| 具名人物 | 马云、许家印等 | "一位企业家""某位商界大佬""当年那位" |
| 政策细节 | "3000亿保障房专项贷款""70余城国资收储" | 删除具体数字，保留方向性描述（"多城市推行""大量资金涌入"） |
| 敏感政策表述 | "政府核价""一房一价" | "核价制度""参考价机制""定价要过审批" |
| 负面定性词 | "炒房客""高位接盘""烂尾楼" | "投资客""接盘""交付困难的项目" |
| 情绪化表达 | "扎心""震惊""万万没想到" | 用具体数据/场景替代，不用情绪词 |
| 绝对化表述 | "彻底""完全""一去不返" | "在发生变化""趋势明显""大不如前" |
| 敏感数据 | 具体破产数字、裁员人数 | "超200家"→"大量"或保留但加"据公开报道" |

### 第二步：风格改写（热点刀锋风格）

**核心转换**：
- 资讯体 → 九边式"我"视角（"我最近跟朋友聊天发现……"）
- 列举式 → 场景式（用身边案例代替罗列数据）
- 结论式 → 追问式（结尾引导评论互动）
- 长段落 → 短段落（适配手机阅读）

**句式替换规则**（参照热点刀锋禁用句式清单）：
- ❌ "说白了" → ✅ 直接说结论
- ❌ "更关键的是" → ✅ "但问题在于""可现实是"
- ❌ "更扎心的是" → ✅ "但问题在于"
- ❌ "你有没有过这种感觉" → ✅ 具体场景描写

### 第三步：字数控制

- 用户说"五百字左右"→ 400-600字
- 用户说"精简"→ 原文的 40-50%
- 未指定 → 原文的 60-70%（去除冗余后）

**精简技巧**：
- 删除重复论述（同一观点说了3遍的，保留最有力的1遍）
- 合并同类段落（如4个"变革"可以合并为2-3个）
- 删除过渡性废话（"接下来我们来看看""综上所述"）

### 第四步：图片提示词生成（可选）

用户说"生成图片提示词"时，提供 3 个风格的英文 prompt：

1. **写实新闻风**（推荐首选）— 摄影风格，直接对应文章场景
2. **概念隐喻风** — 用隐喻/象征表达文章核心观点
3. **数据冲击风** — 信息图/图表风格，适合配合数字标题

每个 prompt 包含：
- 场景描述（what）
- 情绪/氛围（mood）
- 技术参数（16:9, cinematic, editorial 等）

## 输出格式

```
【改写稿】
{改写后的文章}

【改写说明】
| 处理项 | 原文 | 改写后 |
|--------|------|--------|
| ... | ... | ... |

【图片提示词】（如用户要求）
① 写实风：...
② 隐喻风：...
③ 数据风：...
```

## 注意事项

- 改写后的文章必须可以直接发布，不需要用户再修改
- 保留原文核心观点和数据，只改表达方式和敏感词
- 不要过度改写导致原意丢失
- 图片提示词用英文（AI绘图工具普遍英文效果更好）
