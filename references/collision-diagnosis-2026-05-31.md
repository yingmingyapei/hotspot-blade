## 逐篇诊断

### R1-① 多所高校撤销外语学院

**A派（认同）**：早说了学外语没前途 / AI确实厉害 / 我就是转行的
**B派（不认同）**：AI翻译不行，没有文化理解 / 语言不只是翻译 / 我学外语活得挺好

**问题**：文章语气太绝对（"外语已死"），A派占压倒性优势，B派不敢发声。
**改造方向**：不要下"外语已死"的结论，而是问"学外语到底还有没有用？"——给B派（语言爱好者、文科捍卫者）留空间。

### R1-② 胖东来矿泉水停售

**A派（认同）**：胖东来也有今天 / 被神化了就该摔
**B派（不认同）**：一个瓶盖问题至于吗 / 胖东来态度好就行 / 别黑胖东来

**问题**：有对撞潜力（胖东来有大量"信徒"），但文章没有明确攻击他们，只是陈述事实。B派力度不够。
**改造方向**：直接挑战胖东来信徒——"你信任一个品牌到什么程度才叫盲目？"逼B派出来辩护。

### R1-③ 蛋白粉价格暴涨

**A派（认同）**：确实贵了 / 被割韭菜
**B派（不认同）**：？？？

**问题**：没有对立面。谁会为涨价辩护？这是典型的"全民共识型"内容。
**改造方向**：改成"你愿意为健康多花200块吗？"——价格敏感派 vs 健康投资派。或者"国产蛋白能不能替代进口？"——国产派 vs 进口派。

### R1-④ 中日防务争议

**A派（认同）**：说得好！日本就是战败国
**B派（不认同）**：过于强硬 / 外交不应该这样 / 别搞民粹

**问题**：有对撞，但头条用户中"挺日派"极少，A派占绝对优势。
**改造方向**：不要只站在"批判日本"的角度，而是问"强硬外交到底有没有用？"——强硬派 vs 韬光养晦派。

### R1-⑤ 空气炸锅纸致癌

**A派（认同）**：确实贪便宜了 / 三无产品害人
**B派（不认同）**：？？？

**问题**：没有对立面。谁会为三无产品辩护？这是科普文，不是爆款文。
**改造方向**：改成"食品安全该靠消费者辨别还是监管兜底？"——个人责任派 vs 监管责任派。

### R2-① 老人买基金亏70万

**A派（认同）**：银行太黑了 / 老人是受害者
**B派（不认同）**：？？？

**问题**：没有对立面。谁会替银行说话？
**改造方向**：改成"70岁老人应不应该为自己的投资决策负责？"——保护派 vs 责任派。这样B派（"成年人签了合同就该负责"的人）就有话可说了。

### R2-② 存储器暴涨1000%

**A派（认同）**：巨头垄断 / 全产业链被割
**B派（不认同）**：？？？

**问题**：没有对立面。谁会替三星/SK海力士说话？
**改造方向**：改成"你的手机贵了500块，你愿意买国产存储的便宜手机吗？"——国产派 vs 品质派。

### R2-③ 陈丽君被打码

**A派（认同）**：品牌太过分了 / 用完即弃
**B派（不认同）**：品牌有权选择代言人 / 合同到期了可以不续

**问题**：微弱对撞。少数人觉得品牌有权调整策略，但不敢说。文章立场太明确。
**改造方向**：不要直接下"品牌太过分"的结论，而是问"品牌和代言人之间，到底谁欠谁？"——给B派留空间。

### R2-④ 科学无国界科学家有祖国

**A派（认同）**：科学家应该有爱国心 / 钱学森精神
**B派（不认同）**：科学不应该被政治绑架 / 这是道德绑架

**✅ 唯一成功的案例！** 原因：
- 文章给了A派弹药："钱学森回国造导弹"
- 文章给了B派弹药："利用科学的人永远有自己的算盘"
- 文章没有明确站队，让两派自己打
- 结尾"你觉得呢？科学到底有没有国界？"——逼读者选边

### R2-⑤ 住宅变网约房

**A派（认同）**：扰民就应该管 / 法律支持邻居
**B派（不认同）**：房主有权处置自己的房子 / 市场经济

**问题**：理论上有两派，但"支持网约房"的声音太弱。大多数读者都是"被打扰的邻居"视角。
**改造方向**：不要只站在邻居视角，而是问"你的房子你做主，还是邻居说了算？"——产权派 vs 安宁权派。

## 失败模式总结

### 模式1：全民共识型（最常见，6/10篇）

选题本身就没有对立面。所有人都站在同一边（银行黑、涨价可恶、三无产品害人）。

**根因**：选题时只看热度，不看对撞潜力。
**修复**：选题时必须通过"共识型选题过滤器"——如果答不出"谁会反对"，直接跳过。嫁接到真实议题库中的分裂议题。

### 模式2：写法太"正确"（3/10篇）

选题有对撞潜力，但文章立场是"弱者有理、强者有罪"——银行黑、品牌坏、垄断该死。这种立场太安全了，没有人会反对"弱者有理"。

**根因**：写作时只站在"正确"的那一边，没有主动站在"不那么正确"的那一边。
**修复（v3.1修正版）**：写法要足够笃定、不留余地。你越笃定，同意的人越爽，不同意的人越怒。**不要机械列出A派B派观点**——那是辩论赛报名表，不是爆款文章。立场鲜明本身就是最好的"两派弹药"。

### 模式3：结尾太软（10/10篇）

所有文章结尾都是"你怎么看？""你遇到过吗？"——问感受，不逼站队。

**根因**：结尾引导太温和。
**修复**：结尾用笃定的判断句（"归了包堆，就是这么个理儿"）或戳心的反问（"你家老人去银行存钱，有没有被'推荐'过？"）。不是"你是哪一派？"——读者自己会选边。

## 唯一成功案例的写作特征（R2-④ 科学无国界）

1. **立场没有过于极端**：不是单方面下判断，而是展示了两种视角——但这只是因为话题本身太敏感，不适合过于笃定
2. **用了历史案例**：钱学森回国（A派弹药）+ 同一句话指向不同的人（B派弹药）
3. **话题本身天然是50/50**："科学家该不该爱国"这个话题天然分裂——民族主义者 vs 自由主义者
4. **结尾有余味**："你觉得呢？科学到底有没有国界？"

**注意**：这篇文章的"成功"主要是因为选题正确（天然分裂议题），而不是写法完美。实际上，如果写法更笃定（直接说"科学就是有国界"或"科学不应该有国界"），对撞效果会更强。

## 改进方向（v3.1修正版）

1. **选题阶段**：每个选题必须嫁接到真实议题——答得出"谁会反对"才能用，答不出就跳过
2. **写作阶段**：用笃定的语气说一个有争议的观点，用数据撑住它。**不要机械列出两派观点**——立场鲜明本身就是最好的"两派弹药"
3. **结尾阶段**：用笃定的判断句或戳心的反问收尾。不是"你是哪一派"，不是"你怎么看"——读者自己会选边
4. **评分阶段**：议题分裂度权重30%（第一权重），淘汰所有共识型选题
