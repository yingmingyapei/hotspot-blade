# 写作风格集成指南

## 如何将新写作风格融入热点刀锋

当蒸馏出一个新的人物/写作风格后，按以下步骤集成到热点刀锋：

### 第一步：创建写作风格手册

从蒸馏产物（著作、对话、表达DNA）中提炼融合，创建独立的「写作风格手册」。手册必须包含：

1. **文风配方**：词汇成分占比（如文言30%+市井白话50%+方言20%）
2. **句式模板**：3-5个必用句式，可直接套用
3. **开头模板**：2-3种切入方式
4. **中段结构**：逻辑递进方式
5. **结尾公式**：金句/反问句模板
6. **禁忌清单**：绝对不能用的表达

### 第二步：修改热点刀锋 SKILL.md

需要修改的8个位置：

| # | 位置 | 修改内容 |
|---|------|---------|
| 1 | 技能描述 | 增加新风格说明 |
| 2 | 五层密码后 | 新增「XX风格变体」完整章节 |
| 3 | 七步工作流 | 增加新风格的Hook选项 |
| 4 | 钩子设计 | 增加新风格的开头模板 |
| 5 | 标题生成 | 增加新风格的标题公式 |
| 6 | 金句模板库 | 增加新风格的金句 |
| 7 | 句式速查 | 增加新风格的句式清单 |
| 8 | 三宗罪自检 | 增加风格一致性检查 |

### 第三步：更新 cronjob prompt 模板

同步修改 `templates/cronjob-prompt.md`，确保定时任务也能使用新风格。

### 第四步：添加到相关Skill表

在SKILL.md末尾的「相关 Skill」表格中添加新风格技能的引用。

### 风格自动匹配规则

新风格集成后，需要定义其适用场景：
- 社会/民生/职场/就业 → 优先XX风格
- 政策/国际/科技/两岸 → 优先反差互怼
- 情感/生活/感悟 → 优先XX风格
