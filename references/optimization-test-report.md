### Buzzing.cc抓取优化

优化后的正则排除模式（12个）：
```python
exclude_patterns = [
    r'http[s]?://',  # URL
    r'\.buzzing\.cc$',  # 网站域名
    r'www\.[a-z]+\.[a-z]+$',  # www.开头的域名
    r'↑',  # HN Points格式（包含↑符号）
    r'HN\s*Points',  # HN Points文本
    r'^Show\s*HN:',  # Show HN前缀
    r'^I\s+switched',  # 常见开头
    r'^The\s+Old\s+Guard',  # 常见开头
    r'^Buzzing\s*-',  # Buzzing网站标题
    r'^用中文浏览',  # 中文描述
    r'^本站并非官方网站',  # 免责声明
    r'^最后更新于',  # 更新时间
    r'^Twitter\s*@',  # Twitter标签
]
```

## 三、发现的问题及修复

### 问题1：知乎热榜抓取失败（状态码401）

- **原因**: API认证问题，需要登录态
- **修复**: 添加了备用方案（curl_cffi直接抓取），但备用方案也失败
- **解决方案**: 使用`opencli zhihu hot`命令（需要Chrome扩展+登录态）
- **影响**: 不影响技能本身，只是测试环境问题

### 问题2：tophub.today已弃用

- **原因**: 2026-05-07三工具交叉验证确认：IP被Cloudflare风控标记
- **修复**: 从SKILL.md中删除了所有tophub相关代码和描述
- **替代方案**: 使用36氪+HN+IT之家作为科技垂直补充

### 问题3：opencli无xitter命令

- **原因**: opencli v1.7.8没有xitter命令
- **修复**: 从SKILL.md中删除了X trending相关描述
