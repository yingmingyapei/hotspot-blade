## 执行策略

当 Chrome 不可用时：
1. 检测 `opencli doctor` → Extension 未连接
2. 尝试启动 daemon → 仍无法连接（Chrome进程不存在）
3. 切换到备用数据源：百度热搜 + HN Firebase + Buzzing.cc
4. 3个源全部成功才继续，否则停止任务
5. 在执行报告中注明"Chrome未运行，使用备用数据源"

## 与铁律的关系

"铁律"（必须使用opencli）的核心意图是：**不使用百度搜索等经过算法加工的二手数据**。

备用数据源的特殊性：
- 百度热搜（top.baidu.com）是百度的**原始热榜页面**，不是百度搜索结果
- HN Firebase 是 HN 的**官方 API**，不是第三方聚合
- Buzzing.cc 是**直接访问**目标网站，不是通过搜索引擎间接获取

这三者都是"第一手数据"，符合铁律的精神。
