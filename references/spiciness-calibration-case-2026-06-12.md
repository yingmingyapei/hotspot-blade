# 辣度校准实战案例：80万存款70万房贷

> 日期：2026-06-12
> 话题：33岁大厂被裁，存款80w房贷70w，该不该提前还贷？

## 初稿问题诊断

用户反馈："辣度不够，观点不够清新，去除两派争议这些词语"

### 问题1：机械两派结构
```
❌ 评论区吵翻了。一派说还，无债一身轻；另一派说别还，钱拿去理财。
```
**问题**：列出两派观点 = 模板化、生硬、没有作者立场
**修正**：直接下判断，不用预设评论区反应

### 问题2：翻转不够深
```
❌ 这不是选择题，是纠错题。
```
**问题**：翻转有了，但不够扎心
**修正**：用身份重定义——"你的实际身家是10万，不是80万"

### 问题3：辣度不足
```
❌ 更扎心的是什么？80万放银行不动，购买力一年比一年缩水。
```
**问题**：用了"更扎心的是"（已退役句式），且观点不够犀利
**修正**：直接说——"你以为你是'有存款的人'。你不是。你是'帮银行存钱的人'。"

## 终稿核心手法

### 身份重定义（最强辣度来源）
```
你以为你是"有存款的人"。你不是。你是"帮银行存钱的人"。
```
把读者的正面标签（有存款）换成负面标签（帮银行打工），触发身份防御机制。

### 数字暴击（让读者算账）
```
80万存款减去70万房贷，你的实际身家是10万。不是80万，是10万。
```
用简单减法颠覆读者的认知——他以为自己有80万，实际只有10万。

### 笃定判断句收尾
```
这道题的答案不是"该不该还"。是"早该还了"。
```
不留余地，不用"你怎么看"，直接下判断。

## 辣度公式

```
辣度 = 新颖度 × 反讽深度 × 无法反驳度

新颖度：观点是不是大多数人不会说的话
反讽深度：有没有打碎读者的自我合理化
无法反驳度：读者能不能用一句话反驳（越容易反驳，评论越多）
```

## 禁用表述（本案例新增）

- ❌ "一派说X，另一派说Y"
- ❌ "评论区吵翻了"
- ❌ "两派都在算账"
- ❌ "更扎心的是"（已退役）
- ❌ "你怎么看？"（结尾禁用）
