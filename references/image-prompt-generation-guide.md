# 图片提示词生成指南（热点刀锋配套）

> 触发条件：用户说"生成图片提示词"或为微头条配图时使用。
> 注意：只输出文字 prompt，不调用 image_generate 工具。

## 四步法

### Step 1：提取文章核心意象
从文章中找出 3-5 个最具画面感的元素：
- 具体物件（筷子+手机、老照片、红烧肉）
- 场景空间（饭桌、航母甲板、霍尔木兹海峡）
- 情绪基调（温暖vs冰冷、紧张vs平静、怀旧vs现实）

### Step 2：构建场景描述（主体段）
- **开头**：机位+构图（cinematic wide-angle / split-screen / overhead）
- **主体**：具体场景描写，含光线、色调、人物动作
- **细节**：至少 3 个具象细节（品牌、颜色、材质、表情）
- **氛围**：整体 mood（dramatic / emotional / tense / warm）

### Step 3：元素象征表格
将画面中的每个关键元素映射到文章主题的象征意义：

| 元素 | 象征 |
|------|------|
| 左暖右冷分屏 | 过去与现在的温度差 |
| 商人剪影站航母 | 以商止战 |
| 水面漂浮的红印文件 | 签字画押 |

### Step 4：风格关键词
固定输出格式：
```
**风格关键词：** photorealistic, cinematic, [具体光线], [构图方式], [色调], [镜头语言]
```

## 常用视觉隐喻库

| 主题 | 视觉隐喻 |
|------|----------|
| 时间对比 | 分屏（暖色vs冷色）、老照片vs屏幕光 |
| 权力博弈 | 棋盘、签字桌、航母甲板、空旷会议室 |
| 技术入侵 | 蓝白屏幕光vs暖黄灯光、低头族群像 |
| 经济压力 | 天平、空钱包、涨跌曲线背景 |
| 和平/战争 | 夕阳海面、消散的烟雾、握手vs握拳 |
| 怀旧 | CRT电视、老式灯泡、胶片颗粒感 |
| 碎片化 | 破碎的镜子/照片、散落的拼图 |

## 光线速查

| 光线 | 情绪 | 适用 |
|------|------|------|
| Golden hour（暖金） | 怀旧、温暖、史诗 | 回忆、和平、成就 |
| Cold blue-white（冷蓝白） | 孤独、现代、疏离 | 科技入侵、异化 |
| Chiaroscuro（明暗对比） | 戏剧、冲突、悬疑 | 权力、阴谋、抉择 |
| Soft diffused（柔光） | 温柔、日常、真实 | 生活场景、人文 |
| Neon / screen glow | 数字、未来、沉迷 | 手机/网络主题 |

## 注意事项
- 用户偏好：只输出 prompt 文字，不生成图片
- 构图优先用 cinematic / editorial photography 风格
- 中文文章配图倾向写实风格，避免过于抽象
- 场景描述中融入 2-3 个文章中的原文细节（如"红烧肉""霍尔木兹海峡"）
