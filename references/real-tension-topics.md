# 真实议题库：中国社会天然存在的立场分裂

> **核心理念**：议题是客观存在的事实，不是我们刻意打造的。我们的工作不是造刀，是找到那把已经存在的刀，然后亮出来。
>
> **使用方法**：抓到热榜话题后，在此库中找到可以嫁接的真实议题，用热点新闻作为"引信"，引爆已经存在的社会矛盾。

---

## 一、为什么需要真实议题库？

### 虚假分裂 vs 真实分裂

| 维度 | 虚假分裂 | 真实分裂 |
|------|---------|---------|
| 来源 | 作者刻意制造的争议 | 社会客观存在的矛盾 |
| 深度 | 情绪层面，来得快去得也快 | 利益/价值观层面，持续数年甚至数十年 |
| 群体基础 | 没有明确的对立群体 | 两派群体真实存在，人数足够多 |
| 数据支撑 | 没有，只有观点 | 有大量真实数据、案例、政策文件 |
| 可持续性 | 一次性流量 | 每次有新事件都能重新引爆 |
| 读者感受 | "又在标题党" | "终于有人说出来了" |

### 真实议题的三个特征

```
1. 涉及两个明确群体的利益冲突（不是观点不同，是利益不同）
2. 存在"正确但有争议"的价值判断（没有标准答案）
3. 历史上这个话题每次出现都会吵（可验证的持续性）
```

## 二、核心真实议题库（按生命周期排序）

### 🔴 永恒议题（存在10年以上，永远不会过时）

#### 1. 鸡娃 vs 佛系教育
```
A派（鸡娃）：不努力就被淘汰，我是在帮孩子
B派（佛系）：童年被毁了，孩子不快乐有什么用

数据弹药：
- 中国家庭平均教育支出占收入30%+
- 学区房溢价率30-50%
- 青少年抑郁检出率逐年上升
- "双减"政策后课外培训转入地下

嫁接热点类型：
- 教育政策变动（减负、中考改革、高考分流）
- 学区房价格波动
- 名校录取数据
- 青少年心理健康事件

嫁接示例：
热点："14岁以下喜提3天小长假"
→ "孩子放3天假，评论区却吵翻了：到底是好事还是灾难？"
A派弹药：终于减负了！孩子不该被作业压死
B派弹药：3天假谁带？双职工请假扣工资谁来补？
```

#### 2. 彩礼该不该要
```
A派（要彩礼）：传统习俗，是对女方的尊重和保障
B派（不要彩礼）：卖女儿，是陋习，应该废除

数据弹药：
- 全国平均彩礼金额（各地差异巨大）
- 因彩礼分手/离婚的比例
- 最高法关于彩礼返还的司法解释
- "零彩礼"地区的婚姻满意度数据

嫁接热点类型：
- 彩礼纠纷案件
- 各地彩礼政策
- 婚恋市场数据
- 因彩礼引发的社会事件

嫁接示例：
热点："某地规定彩礼不得超过X万"
→ "彩礼限高令来了，评论区两派人吵疯了"
```

#### 3. 老人帮带孩子（隔代育儿）
```
A派（需要帮忙）：双职工没办法，不靠老人孩子谁带？
B派（不想带）：老人没有义务，退休了应该享受生活

数据弹药：
- 中国隔代育儿比例超过50%
- 二胎/三胎后老人带娃压力激增
- 因带娃引发的婆媳矛盾案例
- 托育机构覆盖率不足5%

嫁接热点类型：
- 托育政策
- 老人带娃健康问题
- 婆媳矛盾事件
- 生育政策变动

嫁接示例：
热点："60岁老人带孙子累倒住院"
→ "老人到底该不该帮带孩子？评论区吵翻了"
```

#### 4. 买房 vs 租房
```
A派（买房）：有房才有安全感，租房是给别人还贷
B派（租房）：房贷绑死30年，不如租房自由

数据弹药：

- 房价收入比（北京/上海超过30倍）
- 租房权益保障政策进展

嫁接热点类型：
- 房价波动
- 房贷利率调整
- 租购同权政策
- 烂尾楼/延期交房事件

嫁接示例：
热点："某城市房价跌破XX"
→ "现在到底该买房还是租房？评论区两派人打起来了"
```

#### 5. 全职妈妈 vs 职场妈妈
```
A派（全职）：孩子前3年最关键，陪伴是最好的教育

数据弹药：

嫁接热点类型：

- 托育政策

嫁接示例：
热点："全职妈妈离婚获赔X万家务补偿"
```

## 三、议题嫁接方法论

### 嫁接三步法

```
第一步：拿到热榜话题
第二步：问自己"这个话题背后，藏着哪个真实议题？"
第三步：用热点新闻作为"引信"，引爆那个真实议题
```

### 嫁接成功标志

```
✅ 标题可以用"该不该""对不对""值不值"来提问
✅ 评论区自然形成两派
✅ 两派都有大量拥趸（>20%读者）
✅ 两派都有话可说（反驳成本低）
✅ 这个议题之前已经被讨论过（有历史数据）
```

### 嫁接失败标志

```
❌ 只有一派有话说
❌ 没有明确的对立群体
❌ 读者看完只会说"是的"/"同意"
❌ 这个议题是第一次出现（没有群体基础）
```

## 四、选题时的"天然分裂"快速判断

拿到一个热榜话题，用以下问题快速判断有没有天然分裂：

```
1. 这个话题涉及哪两个群体？（必须答得出）
2. 这两个群体的利益是否冲突？（必须是利益冲突，不是观点不同）
3. 两个群体的人数是否都够多？（都>20%读者）
4. 这个冲突是今天才有的，还是一直存在？（一直存在更好）
5. 我能不能用一句话同时激怒一派和打动另一派？（必须能）
```

如果5个问题都能回答，选它。如果有任何一个答不上来，换话题。

|------|----------|------|
| 科学无国界 | ✅ | 民族主义者 vs 自由主义者 |

### 改造方案

| 原话题 | 嫁接的真实议题 | 改造后标题 |

| 蛋白粉暴涨 | 消费降级 vs 品质生活 | "蛋白粉涨到480一桶，你还喝吗？还是换成国产大豆蛋白？" |
