# 热点刀锋 · 三风格融合指南

> 热点刀锋 v2 = 九边骨架 × 郭德纲皮肤 × 反差互怼钩子

## 风格选择规则

每篇微头条**只用一种主风格**，不要混搭。根据话题类型自动匹配：

| 话题类型 | 主风格 | 理由 |
|---------|--------|------|
| 社会/民生/职场/就业 | **郭德纲** | 市井烟火气，老百姓爱听 |
| 政策/国际/科技/两岸 | **反差互怼** | 需要破防感和争议性 |
| 情感/生活/感悟 | **郭德纲** | 半文半白有质感 |
| 财经/商业/消费 | **九边** | 需要数据支撑和理性分析 |
| 突发热点/争议事件 | **反差互怼** | 需要快速破防 |

## ⚠️ 核心禁忌：风格不混搭

**错误示例**（混搭）：
```
我这人没什么文化，不过说白了，XX这件事的本质是……（九边+郭德纲混搭）
```

**正确示例**（纯郭德纲）：
```
我这人没什么文化，不过有件事儿吧，我琢磨了好些年。古时候有个故事……（纯郭德纲风格）
```

**正确示例**（纯九边）：
```
说白了，XX这件事的本质是……你以为是Y？其实没那么简单。（纯九边风格）
```

## 风格切换时机

- **同一篇内不切换风格**
- **同一批5篇可以不同风格**（第1篇郭德纲，第2篇九边，第3篇反差互怼……）
- **定时任务模式**：根据当天热点话题类型自动选择，5篇可能用2-3种风格

## 三风格核心差异速查

| 维度 | 九边 | 郭德纲 | 反差互怼 |
|------|------|--------|----------|
| **人设** | 理性分析师 | 市井智者 | 争议制造机 |
| **文风** | 极致人话化 | 半文半白 | 犀利挑衅 |
| **切入** | 个人经历/痛点 | 借古讽今/市井类比 | 数字暴击/身份反转 |
| **中段** | 层层递进 | 欲擒故纵 | 暴风雪开场 |
| **收尾** | 清醒通透 | 对仗押韵金句 | 定音锤一句话 |
| **金句** | 「说白了就是……」 | 「归了包堆，就是这么个理儿」 | 「这不叫X，叫Y」 |
