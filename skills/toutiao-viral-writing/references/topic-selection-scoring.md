# 五维选题评分体系

> 来源：2026-06-15 选题报告实战。从5平台197条原始数据中精选5个S级话题。
> 评分体系用于快速筛选话题，决定是否投入写作精力。

---

## 一、五维评分模型

| 维度 | 权重 | 含义 | 9-10分 | 5-6分 | 1-2分 |
|------|------|------|--------|-------|-------|
| 钱包距离 | 35% | 跟读者的钱包有多近 | 直接关乎工资/房价/物价 | 间接影响 | 跟钱包无关 |
| 反驳成本 | 25% | 读者能不能一句话反驳 | 一句话就能反驳 | 需要举例子 | 无法反驳 |
| 物件锚点 | 15% | 有没有读者生活中见过的具体"东西" | 天天用/见 | 知道但不接触 | 纯抽象概念 |
| 头条适配 | 15% | 头条用户爱不爱看 | 社会民生/国际政治 | 科技需降维 | 饭圈/小众 |
| 天然分裂度 | 10% | 天然存在两个对立群体 | 势均力敌 | 需转化 | 纯感动/猎奇 |

**评级：** S级(>=8分) → 优先写 | A级(6-8分) → 可选 | 淘汰(<6分)

**交叉验证加分：** 同一话题在2+平台同时出现 → 额外+0.5分

---

## 二、五维评分与情绪三层术的映射

| 评分维度 | 对应情绪层 | 写作启示 |
|---------|-----------|---------|
| 钱包距离高 | 挑动（翻旧账） | "你当年有机会但没买" |
| 反驳成本低 | 煽动（定罪） | 一句话就能定罪，读者没法反驳 |
| 物件锚点强 | 调动（讲故事） | 用具体物件串起叙事 |
| 天然分裂度高 | 煽动（引战） | 评论区自动分裂成两派 |
| 头条适配高 | 调动（共鸣） | 用户画像匹配，读完率高 |

**核心洞察：钱包距离+反驳成本 = 情绪操控的双引擎。钱包距离决定读者"要不要看"，反驳成本决定读者"要不要评论"。**

## 三、选题来源与优先级

| 平台 | 数据质量 | 九边适配性 | 说明 |
|------|---------|-----------|------|
| 知乎热榜 | 强 | ★★★★★ | 社会议题丰富，矛盾清晰，适合九边剖析 |
| 微博热搜 | 强 | ★★★ | 娱乐/民生混杂，需预过滤；选"民生/职场/社会"类 |
| 百度热搜 | 强 | ★★★★ | 社会民生类话题丰富 |
| 36氪热榜 | 强 | ★★ | 商业/科技为主，需降维 |
| B站热门 | 强 | ★★ | 娱乐/二次元/游戏主导，不适合九边严肃分析 |
| 同花顺问财 | 强 | ★★★★ | 财经+民生资讯搜索，API调用快，数据结构化（需IWENCAI_API_KEY） |
| X/Twitter | 中 | ★★★ | 港台/国际视角，适合找反差素材（需cookie或API） |
| Facebook | 中 | ★★ | 港台用户为主，生活/娱乐类为主 |

### 同花顺问财API搜索（社会民生素材）

当需要快速搜索社会民生类写作素材时，使用news-search技能的iwencai API：

```bash
# 搜索社会民生热点
python3 scripts/news_search.py -q "房价楼市最新"
python3 scripts/news_search.py -q "年轻人就业消费"
python3 scripts/news_search.py -q "结婚生育养老"
```

**有效搜索关键词组合（已验证）：**
- "社会民生热点" — 综合民生政策
- "房价楼市最新" — 房价走势、楼市信号
- "年轻人就业消费" — 就业难、消费降级
- "结婚生育养老" — 人口问题、养老金融

**优势**：返回结构化数据（标题+摘要+来源），比爬取社交媒体更稳定。
**限制**：偏财经视角，纯社会新闻覆盖度不如微博/知乎。

4. 老破小抢手 — 反常识角度（抢手≠好）
5. 丰田章男孤独 — 外国映射（日本=中国XX行业的明天）

## 五、选题报告→微头条的转化流程

```
1. 五维评分筛选 → 选出S级话题
2. 确定情绪方向（无奈/愤怒/讽刺）
3. 匹配情绪三层术（调动/挑动/煽动）
4. 选择开头模板（直陈事实/数字暴击/场景描写）
5. 嵌入九边句式（本质剥离刀/定性重锤/你猜？）
6. 设计台阶+补刀收尾
7. 算法发布检查（标题20-30字、配图、标签、时机）
```
