## 模板一：7段式标准写作模板

在原五步法基础上，嵌入「站队钩子」+「评论区置顶价值指引」，保留犀利感，提升收藏转发和评论深度。

### 固定结构

```
1. 开场第1句：数字暴击（具象数字，带体感）
2. 开场第2句：击穿预期（反常识结论）
3. 开场第3句：悖论提问（勾住往下看）
4. 中段核心：本质剥离刀/定性重锤 + 1个细节数据支撑
5. 中段预埋：站队钩子（两派观点中立摆出来）
6. 收尾：递进王句式 + 行为/回忆式具体追问
7. 评论区置顶：1句可落地行动建议（不放正文）
```

**核心规则**：正文保持纯刀锋，行动建议只放评论区置顶。正文加行动指引会把转发动机从"情绪共鸣"变成"信息分享"，传播力降一个量级。

### 完整示例（主题：年轻人开始批量注销信用卡）

```
去年一年，全国注销了2300万张信用卡。

2300万张。

以前以卡养卡的人，现在都在抢着销户？

这不是大家不爱花钱了，是年轻人终于算明白透支的账了。
分期手续费折成年化利率接近18%，比大部分理财收益高一倍。

有人说销户是成熟了，不碰超前消费；
有人说只是没钱了，想销户保征信。

很明显大家都在往低负债走。但更关键的是——银行偷偷涨利息的时候，从来没提前通知过你。

你手里现在还有几张信用卡？最近有销户的想法吗？
```

**评论区置顶**：说句实在的，真想用信用卡，只留一张日常消费就行，多了全是负债陷阱。

**示例分析**：
- 句式：数字即幽默（第1-2段）+ 定性重锤（第4段）+ 站队钩子（第5段）+ 递进王（第6段）
- 结尾：行为/回忆题（"最近有销户的想法吗"）
- 价值指引：评论区置顶，不污染正文

## 模板二：全链路互动模板

### 2.1 中段站队钩子公式

```
公式：有人说[A观点]，有人说[B观点]。
位置：中段核心论点之后
原则：中立呈现，不带倾向
```

**正反对比**：

```
❌ 差："我觉得这就是智商税"
   → 一言堂，只有认同你的人评论，反对的直接划走

✅ 好："有人说这是花小钱买开心，值；有人说这就是明着割韭菜，纯智商税"
   → 两边都会来留言
```

### 2.2 结尾提问升级公式

从「是非题」升级为「行为回忆题」，评论长度翻倍，算法权重更高。

| ❌ 低互动（是非题） | ✅ 高互动（行为/回忆题） |
|-------------------|----------------------|
| 你买过小金豆吗？ | 你买的第一颗小金豆，是因为什么契机？ |
| 你觉得房价会跌吗？ | 你身边最近买房的人，是抄底了还是观望？ |
| 你用信用卡吗？ | 你第一张信用卡，是上班第几年办的？ |
| 你会提前还贷吗？ | 你身边提前还贷的人，后来后悔了吗？ |
| 你怎么看？ | 你家那边现在多少钱一克？ |

### 2.3 正文预判反对引导（替代小号评论）

**⚠️ 不用小号评论**（关联风险高：同设备/同IP/同支付信息 → 主号被封）。

在正文里预判反对意见，读者会自发替你验证：

```
模板："评论区肯定有人说[反对意见]——但你[反驳/算账]就知道了。"

示例1：
"评论区肯定有人说'我买的金豆就赚了'——
你买的要是周大福的，你去回收试试，一克少你30块，你就知道了。"

示例2：
"评论区肯定有人说'开车方便多了'——
你把油费+停车+保险+折旧加起来，再除以365天，你就知道了。
方便是真方便，贵也是真贵。"
```

**效果**：反对派看到"肯定有人说"会忍不住来反驳，支持派会帮你说话。评论区自动形成对立讨论。零风险，效果比小号更好。

## 模板三：三级选题池搭建模板

### 三级选题池

| 选题层级 | 内容定位 | 选题方向示例 | 发布节奏 |
|----------|----------|--------------|----------|
| 热点级 | 追突发热点，吃流量红利 | 政策调价、行业大事件、热门社会新闻 | 热点出现2小时内出稿 |
| 痛点级 | 永恒日常痛点，流量稳定 | 工资、房贷、通勤、消费套路、职场潜规则 | 每日1-2篇，固定更新 |
| 认知级 | 反常识冷知识，自带传播 | 行业内幕、消费陷阱、生活常识误区 | 每周2-3篇，拉高账号价值 |

### 痛点级常备选题（可反复使用不同切口）

```
□ 工资条上扣的钱到底去了哪
□ 租房vs买房，30年总成本对比
□ 一个孩子从出生到18岁的真实花费
□ 外卖会员到底省没省钱
□ 医保报销后自费部分到底有多少
□ 提前还贷的真实收益计算
□ 开车通勤vs地铁通勤的真实成本
□ 信用卡分期的实际年化利率
□ 充电桩vs加油，电车真的省吗
□ 双十一优惠券背后的定价套路
```

### 认知级常备选题

```
□ 超市里"鲜榨"果汁的真实成本
□ 保质期≠新鲜期，临期食品的真相
□ 你买的"有机"到底有机吗
□ 酒店星级的评定标准漏洞
□ 奶茶店的成本结构（一杯原料成本不到3块）
□ 化妆品的"专柜价"和"出厂价"差多少
□ 电影票房的分账规则（导演能拿多少）
□ 快递费的真实成本（你以为8块其实只要2块）
□ 网红餐厅的翻台率才是利润密码
□ 你每天喝的"矿泉水"可能就是自来水
```

### 风险红线（3类题材绝对不硬杠）

| 红线 | 说明 | 替代方案 |
|------|------|---------|
| 民生政策类 | 医保、社保、公积金、教育医疗，不为反共识唱反调 | 可以算账呈现政策对个人的具体影响 |
| 灾难悲剧类 | 事故、灾害、恶性案件，不玩冷幽默、不找反面角度 | 纯事实+实用信息 |
| 特定群体类 | 军人、医生、教师、基层劳动者，不一竿子打翻 | 对系统/机制提出质疑，不对群体定性 |

## 模板四：单篇五维数据复盘表

每篇发布后6小时对照复盘，精准定位问题。

| 数据指标 | 均值参考 | 低于均值对应问题 | 优化方向 |
|----------|---------|-----------------|---------|
| 点击率(CTR) | <3% 🔴 / 3-5% 🟡 / 5-10% 🟢 / >10% 🔵 | 开场钩子无力，数字没冲击力 | 改前3句话，数字+冲突前置到15字内 |
| 完读率 | <20% 🔴 / 20-40% 🟡 / 40-60% 🟢 / >60% 🔵 | 前300字留不住人，中段拖沓 | 拆长句、砍废话，核心观点提前，补细节数据 |
| 评论率 | <0.5% 🔴 / 0.5-2% 🟡 / 2-5% 🟢 / >5% 🔵 | 结尾提问太空，没有站队空间 | 升级为行为/回忆题，中段补站队钩子 |
| 转发率 | <0.3% 🔴 / 0.3-1% 🟡 / 1-3% 🟢 / >3% 🔵 | 没有可截图的金句 | 打磨1句刀锋金句（本质剥离刀/身份重定义） |
| 关注转化率 | <0.1% 🔴 / 0.1-0.5% 🟡 / 0.5-2% 🟢 / >2% 🔵 | 人设没立住，读者觉得"文章不错但不关注" | 加个人经历、统一调性、带出口头禅 |

### 诊断流程

```
数据出来后，按以下顺序检查：

1. 最差维度是哪个？→ 对应优化模块
2. 展现量正常但CTR极低？→ 问题在标题，不是内容
3. CTR正常但完读率低？→ 问题在前300字结构
4. 阅读量正常但评论为0？→ 问题在互动机制（缺站队钩子/结尾是非题）
5. 有阅读有评论但不涨粉？→ 问题在人设辨识度
```
