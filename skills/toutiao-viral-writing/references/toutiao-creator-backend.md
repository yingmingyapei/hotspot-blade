# 头条创作者后台数据访问指南

## 登录要求

头条创作者后台（`https://mp.toutiao.com/profile_v4/analysis/works-overall/all`）**需要登录**才能访问粉丝数据。

### 登录方式

1. **扫码登录**（推荐）
   - 页面默认显示"验证码登录"标签
   - 需要切换到"扫码登录"标签：点击 CSS class `.web-login-union__login__scan-code__title` 元素
   - 二维码可能失效，需要刷新
   - 扫码方式：今日头条App → 我的 → 左上角"扫一扫"

2. **验证码登录**
   - 输入手机号 + 短信验证码
   - 需勾选用户协议

3. **第三方登录**
   - 抖音登录、QQ登录、微信登录、密码登录

### 二维码刷新问题

- 二维码可能显示"已失效"状态
- 刷新方法：点击"扫码登录"标签切换，或点击二维码图片
- 如果切换标签后仍显示失效，尝试点击页面上其他元素再切换回来

## 数据获取流程

登录成功后可访问：
- 粉丝画像（年龄、地域、兴趣）
- 内容表现（点击率、读完率、互动率）
- 作品整体分析
- 粉丝增长趋势

## 常见问题

- **问题**：页面显示空白或只有登录框
- **原因**：未登录或登录状态过期
- **解决**：扫码登录后刷新页面

- **问题**：二维码扫描后无反应
- **原因**：二维码已失效
- **解决**：刷新页面重新生成二维码
