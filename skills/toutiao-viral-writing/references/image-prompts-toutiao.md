# 微头条配图提示词模板

> 基于 `toutiao-viral-writing` 技能实战沉淀。配图规则：情绪类1张、故事类3张、干货类9张。

---

## 一、数据对比类（数字暴击）

**适用场景：** 价格对比、数量反差、时间线、行业数据

```
一张信息图，深色背景，左侧[颜色A]大字"[数字A] [标签A]"，右侧[颜色B]大字"[数字B] [标签B]"，中间用[箭头/叉号/对比线]连接，下方小字"[核心结论]"。整体风格简洁犀利，适合手机屏幕阅读，九边风格数据可视化。
```

**示例：**
```
一张信息图，深色背景，左侧红色大字"2027 小批量示范"，右侧绿色大字"2030 规模化量产"，中间用时间轴箭头连接，下方小字"全固态电池真正普及：7-10年"。整体风格简洁犀利，适合手机屏幕阅读，九边风格数据可视化。
```

---

## 二、决策路径类（理性选择）

**适用场景：** 选择对比、利弊分析、行动建议

```
一张决策流程图，顶部大字"[主题]"，左侧分支"[选项A]"→ 下方标注"[结果/代价]"，右侧分支"[选项B]"→ 下方标注"[结果/收益]"。中间用金色大字"[追问句]"。简洁明了，适合手机屏幕。
```

**示例：**
```
一张决策流程图，顶部大字"等固态 vs 现在买"，左侧分支"干等7-10年"→ 下方标注"油车贬值+高油费+老技术"，右侧分支"现在买成熟电车"→ 下方标注"5年后换固态=完美衔接"。中间用金色大字"你猜谁亏？"。简洁明了，适合手机屏幕。
```

---

## 三、金句封面类（情绪张力）

**适用场景：** 单张封面图、观点输出、九边式定性

```
一张深色背景信息图，顶部[颜色]大字"[九边式金句]"，中间[时间轴/数据/对比元素]，底部一行白色小字"[补充说明/追问]"。九边风格，犀利直接，适合微头条封面。
```

**示例：**
```
一张深色背景信息图，顶部红色大字"等固态电池再买电车？说真的，并没有。"，中间时间轴显示"2027小批量→2030规模化→真正普及7-10年"，底部一行白色小字"你一直等，等于一直用老技术、承担更高油费"。九边风格，犀利直接，适合微头条封面。
```

---

## 四、配图数量规则

| 内容类型 | 图片数量 | 说明 |
|----------|----------|------|
| **情绪类** | 1张 | 封面图，金句+情绪张力 |
| **故事类** | 3张 | 叙事感，按时间线/情节推进 |
| **干货类** | 9张 | 信息密度，数据可视化为主 |

---

## 五、配色建议

| 元素 | 推荐颜色 | 用途 |
|------|----------|------|
| **警示/负面** | 红色/橙色 | 数据对比中的负面项、警告 |
| **正面/推荐** | 绿色/蓝色 | 数据对比中的正面项、建议 |
| **文字** | 白色 | 深色背景上的正文 |
| **强调/追问** | 金色/黄色 | 金句、核心追问 |
| **背景** | 深灰/黑色 | 九边风格统一底色 |

---

## 六、尺寸建议

- **比例：** 16:9 或 3:4（手机屏幕适配）
- **分辨率：** 1080p 以上（平台推荐）
- **文字量：** 每张图不超过30字，数字要醒目

---

## 七、九边风格配图核心原则

1. **数字醒目：** 数据是CTR触发器，数字要大、要对比
2. **结论先行：** 第一张图就要让用户知道核心观点
3. **留白感：** 不要堆砌，九边风格是"少即是多"
4. **追问收尾：** 最后一张图可以放互动追问句
