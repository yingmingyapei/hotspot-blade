# 五维数据驱动复盘体系

> 盲目练习20篇不如精准复盘5篇。哪项数据差就改对应模块。

---

## 一、五维复盘对照表

| 维度 | 指标来源 | 诊断方向 | 优化模块 |
|------|---------|---------|---------|
| 点击率(CTR) | 展现量→阅读量 | 标题/开场不够抓人 | Step 2 钩子设计 + Step 6 标题规则 |
| 完读率 | 阅读量→读完量 | 中段留不住人 | Step 3 草稿写作（前300字结构） |
| 评论率 | 阅读量→评论数 | 结尾没引导互动 | Step 3.5 互动机制设计 |
| 转发率 | 阅读量→转发数 | 缺少截图级金句 | Step 7 润色加固 |
| 关注转化率 | 阅读量→新增粉丝 | 人设没立住 | 人设差异化规则 |

---

## 二、逐维诊断清单

### 维度1：点击率(CTR)

```
CTR = 阅读量 / 展现量

< 3%  🔴 标题严重问题
3-5%  🟡 标题需优化
5-10% 🟢 正常
> 10% 🔵 爆款潜力
```

**诊断问题**：
- □ 标题里有数字吗？（没有→加数字）
- □ 标题有反差/冲突吗？（没有→加反差词）
- □ 前15字是否包含了核心冲突？（没有→首行压缩）
- □ 标题风格是否和上一篇重复？（是→换一种风格）
- □ 是否用了被折叠的热词？（检查头条违禁词列表）

### 维度2：完读率

```
完读率 = 读完人数 / 阅读人数

< 20%  🔴 内容留不住人
20-40% 🟡 中段有问题
40-60% 🟢 正常
> 60%  🔵 内容质量高
```

**诊断问题**：
- □ 前300字是否有完整的"钩住→展开→点题"循环？（没有→重写前300字）
- □ 中段句子是否太长？（超过40字/句→拆短）
- □ 中段是否连续3段以上没有数据/细节？（是→补数据）
- □ 段落节奏是否均匀？（是→打乱，长短交错）
- □ 是否有"废段"（不推进论点的段落）？（有→删掉）

### 维度3：评论率

```
评论率 = 评论数 / 阅读人数

< 0.5%  🔴 互动钩子缺失
0.5-2%  🟡 结尾提问太泛
2-5%    🟢 正常
> 5%    🔵 引发讨论
```

**诊断问题**：
- □ 结尾是否提了"是非题"？（是→改为行为/回忆题）
- □ 中段是否有站队钩子？（没有→加"有人说A，有人说B"）
- □ 是否预判了反对意见？（没有→加"评论区肯定有人说……"）
- □ 结尾追问是否具体到普通人能回答？（"你怎么看"→"你家那边多少钱"）

### 维度4：转发率

```
转发率 = 转发数 / 阅读人数

< 0.3%  🔴 缺少传播点
0.3-1%  🟡 金句不够锋利
1-3%    🟢 正常
> 3%    🔵 自传播力强
```

**诊断问题**：
- □ 有没有一句话让人想截图？（没有→补一句本质剥离刀/身份重定义）
- □ 金句是否在前300字内出现？（不在→前移）
- □ 是否有具体的数字对比让人想分享？（没有→加一组反差数字）

### 维度5：关注转化率

```
关注转化率 = 新增粉丝 / 阅读人数

< 0.1%  🔴 人设没立住，读者觉得"文章不错但不关注"
0.1-0.5% 🟡 内容好但缺乏辨识度
0.5-2%   🟢 人设有辨识度
> 2%     🔵 强人设
```

**诊断问题**：
- □ 这篇文章有没有"只有我能写出来"的东西？（没有→加个人经历/独家信源）
- □ 是否体现了固定的思维方式身份？（没有→检查人设一致性）
- □ 是否自然带出了表达标签？（没有→补一句口头禅）
- □ 情绪调性是否和上一篇一致？（不一致→统一风格）
- □ 是否有评论区互动（回复读者评论）？（没有→发布后1小时内回复3-5条）
