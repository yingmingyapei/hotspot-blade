# 爆款刀锋

> **九边式犀利写作模板** — 结论先行 · 数据碾压 · 冷幽默补刀 · 一刀入魂

---

## 人设差异化规则

> 没有人设，所有优化都是给别人的嫁衣。读者关注的是"这个账号"，不是"这篇文章"。

### 1. 固定思维方式身份（不固定职业身份）

❌ 固定职业身份（"我是宝妈/打工人/实体店老板"）→ 限制选题范围
✅ 固定思维方式身份 → 不限选题，但有辨识度

**可选身份模板**（选一个，长期坚持）：

| 身份 | 核心表达 | 适配选题 | 示例口头禅 |
|------|---------|---------|-----------|
| 凡事算账的人 | 用数字替代判断 | 所有钱相关话题 | "我帮你算了一笔账" |
| 爱拆包装的人 | 揭示成本结构/规则漏洞 | 消费、商业、行业 | "翻过来看配料表" |
| 喜欢反着想的人 | 逆向思维解构共识 | 社会现象、热点 | "大家都这么想，但你反过来想" |
| 观察细节的人 | 用小场景切入大话题 | 生活、消费、职场 | "你注意过没有" |

**选择标准**：这个身份能不能覆盖你80%的选题？能不能让你写世界杯也写房价也写小金豆？

### 2. 植入1-2个表达标签

每篇自然带出1次，不强塞，不刻意：

```
✅ 自然植入："我帮你算了一笔账——小金豆一克比大盘贵117块。"
❌ 刻意堆砌："我帮你算了一笔账。说句实在的，我帮你算了一笔账。"
```

**可选标签库**：
- "我帮你算了一笔账"
- "翻过来看"
- "你注意过没有"
- "掰开看看"
- "说句掏心窝的"
- "你品品这个数字"

### 3. 统一情绪调性

选一种主风格，不要一篇犀利一篇温情：

| 调性 | 特征 | 适配身份 |
|------|------|---------|
| 冷静扎心派 | 不带情绪词，纯数据说话，数字本身就是刀 | 凡事算账的人 |
| 接地气吐槽派 | 口语化、带点牢骚、像邻居聊天 | 观察细节的人 |
| 反常识颠覆派 | 每篇都在颠覆一个"常识" | 喜欢反着想的人 |
| 拆解派 | 像拆快递一样拆开行业/产品/现象 | 爱拆包装的人 |

**核心原则**：读者关注你之后，期待的是"可预期的体验"。今天犀利明天温情，粉丝会流失。

## 核心公式

```
传播力 = 点击率（悖论开场 + 数字暴击）
        × 读完率（短句节奏 + 九边式递进）
        × 互动率（自问自答 + 具体追问）
```

## 七步工作流

### Step 1：选题定位

→ 加载 `toutiao-topic-engine` 选题方法论
→ **先判断选题级别**：热点级 / 痛点级 / 认知级（详见 `toutiao-topic-engine/references/three-tier-topic-pool.md`）

| 级别 | 触发条件 | 写法核心 |
|------|---------|---------|
| 热点级 | 热榜出现相关话题 | 反共识 + 悖论开场 + 本质剥离刀 |
| 痛点级 | 日常永恒痛点（工资/房贷/通勤/带娃/消费套路/医疗） | 共情扎心 + 算账 |
| 认知级 | 反常识冷知识（消费真相/数字反差/规则漏洞） | 反常识开场 + 冷知识递进 |

→ 找矛盾 + 定靶子 + 定情绪方向
→ 输出：一句话矛盾定位（含九边式定性）
→ **反共识红线检查**：不碰国家民生政策否定、不碰灾难悲剧玩幽默、不碰特定群体一竿子打翻

**数据铁律**：热榜数据必须当天，排除过期信息，非交易日注明日期。

### Step 2：钩子设计

→ 加载 `toutiao-sentence-patterns` 句式库
→ 开场：数字暴击 + 悖论钩子
→ 中段：本质剥离刀 + 数据跟进 + 自问自答
→ 结尾：递进王 + 预言 + 具体追问

### Step 3：草稿写作

**暴风雪开场**（前100字）：
- 第一行出数字（算法CTR识别）
- 第二行击穿正常预期
- 第三行揭示悖论事实
- 节奏：短句切断，不给喘息

**胖大山打脸**（中段）：
- 至少用两个句式（本质剥离刀 + 定性重锤）
- 数字跟进，数据即幽默
- 短句留白

**递进收尾**（结尾）：
- 递进句把情绪再推一层
- 预言钩子让读者带着"后来呢"离开
- 结尾追问具体到有话说

### Step 3.5：互动机制设计

> 互动率是算法推荐的核心权重。只靠结尾一个问题远远不够。

#### 3.5a 中段预埋「站队钩子」

在本质剥离刀/定性重锤之后，加一句两极分化的中立判断，天然划分阵营：

```
模板："有人说A，有人说B。"
位置：中段核心论点之后
效果：读者自发站队，评论区变辩论场

示例（房价话题）：
有人说房价跌了正好上车，有人说还会继续跌。
你猜评论区谁赢？反正不管谁赢，你都得多看两年。

示例（小金豆话题）：
有人说攒小金豆就是图个仪式感，一百块买开心值；
有人说这就是明着割韭菜，纯纯智商税。
```

**核心原则**：站队钩子必须是中立呈现，不带倾向。你一旦选边，另一派就不评论了。

#### 3.5b 结尾提问升级：从「是非题」到「行为/回忆题」

| 原版（是非题） | 升级版（行为/回忆题） | 为什么更好 |
|---------------|---------------------|-----------|
| "你买过小金豆吗？" | "你买的第一颗小金豆，是因为什么契机？" | 引导讲经历，评论更长 |
| "你怎么看？" | "你家那边现在多少钱一克？" | 具体到能回答 |
| "你会提前还贷吗？" | "你身边提前还贷的人，后来后悔了吗？" | 引导讲故事 |
| "你支持吗？" | "你遇到过类似的情况吗？当时怎么处理的？" | 激活个人经历 |

**评论权重公式**（头条算法）：评论字数 × 互动深度 > 评论数量。一条100字的评论比十条"是的"加权更高。

#### 3.5c 正文预判反对引导（替代小号评论）

不要用小号引导评论（关联风险高）。在正文里预判反对意见，读者会自发替你验证：

```
模板："评论区肯定有人说[反对意见]——但你[反驳/算账]就知道了。"

示例：
"评论区肯定有人说'我买的金豆就赚了'——
你买的要是周大福的，你去回收试试，一克少你30块，你就知道了。"

"评论区肯定有人说'开车方便多了'——
你把油费+停车+保险+折旧加起来，再除以365天，你就知道了。
方便是真方便，贵也是真贵。"
```

**效果**：反对派看到"肯定有人说"会忍不住来反驳，支持派会帮你说话。评论区自动形成对立讨论。

### Step 4：共鸣适配

- 九边骨架 + 普通人血肉
- 替换专业词为普通人感知
- 检查追问是否具体（"你家那边呢？" > "你怎么看？"）

**价值指引规则**（评论区置顶，不放正文）：

正文保持纯刀锋（传播力靠"只戳不救"）。实用价值放评论区置顶：

```
正文：纯刀锋，不掺水
评论区置顶：1句实用建议

示例（小金豆）：
正文写的是"一克贵117块，你不是在投资，你是在交智商税"
评论区置顶："真想买黄金攒钱，直接选银行投资金条，别碰带工艺的首饰款。"
```

**为什么这样做**：读者转发犀利文章的动机是"终于有人说出来了"。正文加了行动指引，转发动机从"情绪共鸣"变成"信息分享"，传播力降一个量级。评论区置顶既保传播力，又拉收藏和关注，两全。

**评论区置顶模板**：
```
"说句实在的，真想[实用动作]，直接[具体做法]，别碰[常见坑]。"
```

**示例**：
```
"说句实在的，真想买黄金，直接去银行买投资金条，别碰首饰款。"
"说句实在的，真想省通勤费，地铁月卡+共享单车，别开车了。"
"说句实在的，真想提前还贷，先看你的贷款利率和国债利率差多少。"
```

### Step 5：钩子强化检查

```
□ 开场前100字有数字？
□ 中段有本质剥离刀或定性重锤？
□ 中段有"你猜？"句式？（≤2次）
□ 结尾有递进句或随机化收尾？
□ 结尾追问具体到让普通人有话说？
□ 有一句话让人想截图转发？
□ 避开了"综上所述""值得注意的是"？
□ 句式频率未超限？
```

### Step 6：算法发布检查

**标题规则**：
- 20-30字（少于20字不够，超过30字被折叠）
- 必须有数字或反差词
- 从7种风格随机选1种：问句/金句/数据反差/短句/讽刺/直陈/时间跨度
- 禁止AI味词汇（"说白了""底层逻辑"）
- **首行极致压缩**：把「数字+冲突」压缩在前15字内（手机单屏可看完的长度）
  - ✅ "1克小金豆，贵了117块"
  - ❌ "最近很多人开始攒小金豆了，但你有没有想过一个问题"

**前300字结构模板**（决定完读率的关键窗口）：

```
第1行：数字暴击（抓眼球）
第2-3行：悖论事实（制造认知冲突）
第4-6行：一个具体场景/细节（让读者"看到画面"）
第7行：本质剥离刀或身份重定义（点题）
```

300字以内完成一次完整的"钩住→展开→点题"循环。读者过了这300字，完读率就稳了。

**分段节奏规则**：
- 开场：每句话单独成段，3段快速抓眼
- 中段：2-3句话一段，松紧交替，避免一屏全是字
- 结尾：提问单独成段，醒目易回复

**热词自然埋入**：
把头条当下的领域热词自然融入内容，提升关键词匹配度，扩大推荐量。
- 热词来源：头条热榜、微头条话题标签
- 植入方式：在中段论点中自然带出，不生硬插入
- 示例："消费降级"、"现金为王"、"性价比替代"

**发布时机**：晚8-10点最佳 > 早7-9点 > 午12-1点

**黄金15分钟策略**（冷启动决定初始流量池）：

发布后15分钟内，点击率和互动率的权重远高于后面的时间段。

```
发布后立即操作：
1. 主号发一条评论（不是小号）——内容是正文里没展开的一个观点
   示例："正文没说的是，回收的时候还有折旧费，一克再扣你5块。"
   效果：制造"作者在评论区补充了更猛的内容"的效应
2. 15分钟内不要编辑/删除/重发——任何操作都会重置冷启动
3. 观察初始推荐量——如果15分钟内展现量<500，检查标题是否有违禁词
```

**发布后6小时监测**：
- CTR < 3% → 换标题重发
- 读完率 < 40% → 检查中段（前300字是否留住了人）
- 互动为0 → 主号发评论引导（不用小号）

### Step 7：润色加固

```
保留：本质剥离刀、定性重锤、递进王、短句节奏、数字排比、具体追问
删掉："综上所述""值得注意的是""值得注意"
打破：均匀段落，长短交错
分散：每段最多1个句式
```

**三宗罪自检**：
1. 细节模糊？→ 补型号/价格/场景
2. 逻辑跳脱？→ 加过渡句
3. 碎句堆砌？→ 合并碎片保留节奏

## 政治敏感性处理

**事实性指代 ≠ 敏感评价**

| 类型 | 示例 | 处理 |
|------|------|------|
| 国家名称 | "中国""美国" | ✅ 正常使用 |
| 中性制度描述 | "筛选管理者的机器" | ✅ 正常使用 |
| 批评性评价 | "体制不行" | ❌ 替换或删除 |
| 政策负面定性 | "有人在替你做主" | ⚠️ 慎用 |

用户说"去除政治敏感性"时，只去除评价性语言，保留事实性指代。

## Pitfalls

### 1. 停在第三步就交付
草稿完成后必须走完 Step 4-7。停在第三步等于交半成品。

### 2. 标题自动化执行纪律条目过多
纪律12-44为高度重复内容。详见 `references/toutiao-skill-quality-fix-guide.md`。

### 3. 技能库不一致
部分技能在 `skills_list` 可见但 `skill_view` 无法访问。详见 `references/skill-library-inconsistency.md`。

### 4. 政治敏感性误判
不要把"中国""美国"等国名也抹掉。只去除评价性语言。

### 5. 引用哲学家/名人原话（2026-06-17教训）
**问题**：在微头条中出现"罗素说""某哲学家说过""尼采认为"等引述字眼，读起来像学术论文而非九边式犀利评论。
**防御**：思想弹药库（包括 russell-flip-arsenal 等）的思想可以使用，但必须**消化成自己的话**。所有哲学/经济学/社会学思想，直接用自己的语言表达，不出现任何"XX说""XX认为"的引述。读者要的是你的判断，不是名言集锦。
