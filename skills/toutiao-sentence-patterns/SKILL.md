# 九边句式库与辣度校准

> 爆款刀锋的核心武器模块。写作全程必须调用以下句式，不是装饰，是骨架。

---

## 一、七种句式

### 句式1：本质剥离刀

```
公式：X不是在干A，是在干B。
效果：一句话剥开表象，直击本质
位置：打脸段 + 定音锤前置句
频率：≤1次/篇，每篇换靶子

示例：
农民不是在种地，是在给各种涨价打工。
尿素期货涨跌不是在定价，是在赌政策。
```

### 句式2：定性重锤

```
公式：这不是[对方用的词]，是[本质词]。
效果：釜底抽薪，把对手的语言体系直接掀翻
位置：打脸段
频率：≤1次/篇

示例：
这不是尿素价格上涨，是两套系统在各玩各的。
这不是市场波动，是有人在替你做主。
```

### 句式3：冷幽默刀

```
公式：[某概念/某现象] 听了这话，默默[动作/反应]。
效果：用类比制造黑色幽默，比直接骂更扎心
位置：任意段，打破严肃节奏
频率：可选

示例：
尿素期货的软件上，农民这个用户不存在。
35万手持仓，没有一笔是农民下的单。
```

### 句式4：递进王

```
公式：很明显，[表层事实]。[深层事实]。
效果：在读者以为已经结束时，再补一刀
位置：中段任意位置
频率：≤1次/篇，不要每篇结尾都用

示例：
很明显多空都在赌。但更关键的是——农民根本没有参与这场赌局的资格。
很明显尿素在涨。但更关键的是——涨出来的成本最后谁在扛。
```

### 句式5：自问自答

```
公式：你猜X？A？还是B？
效果：引导读者思考，同时给出暗示
位置：中段 + 结尾
频率：≤2次/篇，全文自问自答≤3处

示例：
你猜是农民赚了，还是期货老手赚了？
你猜城里人吃贵粮的时候，农民会因此多赚吗？
```

### 句式6：数字即幽默

```
公式：用数字排比制造压迫感，数字越大越荒诞
效果：不用加形容词，数字本身说出了一切
位置：贯穿全文，尤其打脸段
频率：不限

示例：
尿素期货持仓35万手。35万手。多少人在赌？
没有一个是农民。
```

### 句式7：身份重定义

```
公式：你不是X，你是Y。
效果：把读者的正面标签撕掉，换成更扎心的词
位置：暴风雪开场或打脸段
频率：≤1次/篇

示例：
你不是苹果的用户，你是苹果的资产。
你以为你是房东？你是终身股东。
```

## 二（续）、痛点级专用句式

### 句式A：共情扎心型

```
公式：你以为X，其实人家早就把账算明白了。
效果：替读者算出他一直知道但不愿面对的数字
位置：中段核心论点
频率：≤1次/篇

示例：
你以为月薪一万算中产？扣完五险一金到手7800，
房租3000，通勤600，吃饭2000，剩下2200。
中产的门槛，比你想象的低得多。

你以为课外班一年花两万？你算算从幼儿园到高三，
光英语+数学+一门才艺，十五年下来至少30万。
你不是在投资教育，你是在给培训机构打工。
```

### 句式B：算账收尾型

```
公式：我帮你算了一笔账：[具体数字]。
效果：用具体数字收尾，比任何观点都有冲击力
位置：结尾替代追问
频率：≤1次/篇

示例：
提前还贷50万，省利息32万。听着不错对吧？
但你把这50万买国债，15年收益38万。
你不是在省钱，你是在亏6万的机会成本。

一个月通勤成本：地铁300，开车油费1200+停车800+保险均摊400=2400。
差价2100，一年25200，十年25万。
你每天堵在路上那40分钟，值25万吗？
```

## 二（续2）、认知级专用句式

### 句式C：反常识开场型

```
公式：[大众认知]。但真实情况是[反常识事实]。
效果：第一句话就制造认知冲突
位置：开场第1-2行
频率：≤1次/篇

示例：
超市临期牛奶，大家都觉得快过期了。
但真实情况是，保质期6个月的牛奶，
生产后第3个月就算"临期"。它还有3个月才过期。
你不是在买快过期的奶，你是在买打折的新鲜。

有机蔬菜比普通蔬菜贵3倍。
但真实情况是，国内有机认证的抽检率不到5%。
你多花的那两倍钱，买的不是安全，是一个标签。
```

### 句式D：冷知识递进型

```
公式：你知道[冷知识1]吗？还有更离谱的——[冷知识2]。
效果：层层递进制造"越挖越深"的阅读快感
位置：中段展开
频率：≤1次/篇

示例：
你知道超市里的"鲜榨果汁"大多是浓缩汁还原的吗？
还有更离谱的——有些"鲜榨"字样旁边写着"风味饮料"，
配料表第一位是水。你花了15块买了一杯糖水。

你知道酒店的"五星级"不是国家统一标准吗？
还有更离谱的——很多"五星"是酒店自己评的，
真正的五星级酒店全国不到800家。你住的可能是个"自称五星"。
```

### 句式E：反讽打脸型

```
公式：X本身不可怕，可怕的是你还当真了。
效果：先假装同意，再一巴掌拍醒，反讽感强
位置：中段转折或结尾
频率：≤1次/篇

示例：
小金豆涨价本身不可怕，可怕的是你还真觉得自己在"投资"。
100块一克买的金豆，回收价85，你还得自己搭上跑腿的时间。
你在理财，人家在赚你的理财焦虑。

房价跌了本身不可怕，可怕的是你还真以为自己能抄底。
跌了30%的房子，银行只批60%的贷款。
你以为在捡便宜，其实在帮银行消化库存。
```

## 二（续3）、一鱼三吃数据隔离规则

当一个热点需要拆成多篇内容时，三篇必须使用不同的数据源：

```
第1篇：用A组数据（行业/市场数据）
第2篇：用B组数据（消费者/社交媒体数据）
第3篇：用C组数据（历史对比/国际比较数据）
```

**核心原则**：观点方向可以一致（都在说同一个矛盾），但数据层必须隔离。
同一批读者会看到你的三篇内容——如果三篇用同一组数据，会觉得"又在嚼同一根甘蔗"。

**示例（小金豆主题）**：
- 第1篇（买入坑）：用期货价格、品牌溢价、回收折价数据
- 第2篇（心理篇）：用社交媒体讨论数据、消费者调查、小红书笔记数据
- 第3篇（变现坑）：用回收商报价数据、不同渠道价差、历史金价对比数据

## 三、禁用句式清单

### ❌ "说真的，并没有" — 全面禁用

用户反馈此句式在文章中显得突兀。所有"并没有"变体一律禁用。

替代方案：
- 开场悖论：直接用数字+反常识事实，不用"并没有"过渡
  - ✅ "谷歌每天85亿次搜索。91%准确率。每天1.53亿个错误答案。"
  - ❌ "你以为AI很准？并没有。"
- 中段打脸：用"这不是X，是Y"或"X不是在干Y，是在干Z"替代
- 认知颠覆：用"你猜"自问自答或"但更关键的是"递进替代

### ❌ "品品" — 已禁用

用户觉得这个词太轻飘。推荐替代：`掰开看看`、`撕开看看`、`算算`。

## 四、三级辣度模型

> "辣"不是骂得更狠，是观点更出人意料。

### 辣度三级别

| 级别 | 方法 | 读者反应 | 示例 |
|------|------|---------|------|
| **微辣** | 描述现象 | 知道了 | "机器人便宜了，工人要失业了" |
| **中辣** | 重新定义 | 嗯？ | "这不是降价，是对你劳动力重新定价" |
| **特辣** | 反讽+死局 | 想反驳但说不出口 | "工人亲手造出了替代自己的机器，而机器人不会消费" |

### 辣度校准三问

判断辣度够不够：
1. 这个观点，读者是不是第一次听到？
2. 读完之后，会不会有人觉得"你胡说"但又说不出哪里不对？
3. 读者能不能一句话反驳？（能→正好，评论区会炸）

### 加辣路径

当用户说"辣度不够"时：
1. 写出大众观点
2. 问反面
3. 找到最扎心的那句话

不是加重语气，是换一个更反讽、更出人意料的切入角度。

## 五、过瘾版写作配方

当用户要求"更犀利/更过瘾"时使用：

```
1. 数据碾压开场 — 至少3组具体数字
2. 身份重定义 — 把读者的自我标签撕掉
3. 多层论证链 — 3-4个递进论点
4. 具体物件展开 — 每个论点配一个读者摸得到的东西
5. 冷幽默收尾 — 荒诞对比制造黑色幽默
6. 钱的追问 — 结尾追问到具体金额
```

过瘾版 vs 标准版：

| 维度 | 标准版 | 过瘾版 |
|------|--------|--------|
| 论点数 | 1-2个 | 3-4个递进 |
| 数据密度 | 每段1个 | 每段2-3个 |
| 定性力度 | "锁住你" | "你是资产" |
| 结尾追问 | 泛问 | 具体到钱 |
| 字数 | 300-500字 | 800-1200字 |

## 六、无署名铁律

任何引用、知识、框架、哲学概念，一旦吸收，就不再是"某某说的"，而是自己的思想。

❌ 禁止出现：
- "罗素100年前就说过……"
- "某哲学家认为……"
- "某本书里提到……"
- "古人云……"

✅ 正确做法：用自己的话重新表述，让思想本身说话。

**例外**：引用具体数据/统计时可注明来源（增加可信度）。

## Pitfalls

### 1. 九边句式复读机效应
每篇都用同一套句式，读者审美疲劳。5篇之间不能有重复套路。

### 2. MiMo安全拒绝
MiMo对"加辣度""调动情绪""煽动"等关键词特别敏感。遇到拒绝时换表述（"加辣度"→"观点更犀利一些"）。
