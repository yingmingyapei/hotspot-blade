# 三级选题池体系

> 摆脱热点依赖，建立稳定流量基本盘。

---

## 一、三级选题池定义

| 级别 | 名称 | 流量特征 | 出稿节奏 | 写法核心 |
|------|------|---------|---------|---------|
| 热点级 | 突发热点 | 短期爆发，衰减快 | 热点后2小时内 | 反共识 + 数字暴击 |
| 痛点级 | 日常永恒痛点 | 长尾稳定，持续出量 | 随时可发 | 共情 + 算账 |
| 认知级 | 反常识冷知识 | 自带传播力，易出圈 | 每周2-3篇 | 反常识 + 意外感 |

---

## 二、热点级选题（原有体系，简述）

触发条件：热榜出现与账号定位相关的话题。
写法：反共识 + 悖论开场 + 本质剥离刀。
时效：发布后2小时内为黄金窗口。

---

## 三、痛点级选题池

### 核心理念

痛点级不靠追热点，靠"人人都痛但没人算清楚"。流量来源是搜索+推荐双引擎，生命周期可达数月。

### 六大永恒痛点赛道

| 赛道 | 典型话题 | 钱包距离 | 适配句式 |
|------|---------|---------|---------|
| 工资/收入 | 年终奖到手少一半、副业真实收入 | ★★★★★ | 共情扎心 + 算账 |
| 房贷/租房 | 提前还贷亏不亏、房租占收入比 | ★★★★★ | 算账 + 反常识 |
| 通勤/出行 | 开车vs地铁真实成本、油价涨跌 | ★★★★☆ | 算账 + 冷幽默 |
| 带娃/教育 | 课外班真实花费、学区房性价比 | ★★★★★ | 算账 + 共情扎心 |
| 消费套路 | 会员自动续费、大数据杀熟 | ★★★★☆ | 本质剥离 + 反讽 |
| 医疗/保险 | 医保报销真实比例、重疾险坑 | ★★★★☆ | 算账 + 反常识 |

### 痛点级专用句式库

#### 句式A：共情扎心型

```
公式：你以为X，其实人家早就把账算明白了。
效果：替读者算出他一直知道但不愿面对的数字
位置：中段核心论点

示例：
你以为月薪一万算中产？扣完五险一金到手7800，
房租3000，通勤600，吃饭2000，剩下2200。
中产的门槛，比你想象的低得多。

你以为课外班一年花两万？你算算从幼儿园到高三，
光英语+数学+一门才艺，十五年下来至少30万。
你不是在投资教育，你是在给培训机构打工。
```

#### 句式B：算账收尾型

```
公式：我帮你算了一笔账：[具体数字]。你品品。
效果：用具体数字收尾，比任何观点都有冲击力
位置：结尾替代追问

示例：
提前还贷50万，省利息32万。听着不错对吧？
但你把这50万买国债，15年收益38万。
你不是在省钱，你是在亏6万的机会成本。

一个月通勤成本：地铁300，开车油费1200+停车800+保险均摊400=2400。
差价2100，一年25200，十年25万。
你每天堵在路上那40分钟，值25万吗？
```

### 痛点级写作公式

```
开场：一个具体场景（"上周五加班到9点，打车回家87块"）
中段：共情扎心 + 算账（用句式A把痛说清楚）
收尾：算账收尾（用句式B给出一个让人想截图的数字）
```

### 痛点级选题清单（常备池）

以下话题永不过时，可反复使用不同切口：

```
□ 工资条上扣的钱到底去了哪
□ 租房vs买房，30年总成本对比
□ 一个孩子从出生到18岁的真实花费
□ 外卖会员到底省没省钱
□ 医保报销后自费部分到底有多少
□ 提前还贷的真实收益计算
□ 开车通勤vs地铁通勤的真实成本
□ 信用卡分期的实际年化利率
□ 充电桩vs加油，电车真的省吗
□ 双十一优惠券背后的定价套路
```

## 四、认知级选题池

### 核心理念

认知级靠"你不知道吧，其实……"制造意外感。读者看完觉得"涨知识了"，天然想转发给朋友。传播路径是社交分享，不是情绪共鸣。

### 四大认知级赛道

| 赛道 | 典型话题 | 适配句式 |
|------|---------|---------|
| 消费真相 | 超市临期食品、成本结构、品牌溢价 | 反常识开场 + 本质剥离 |
| 数字反差 | 你以为很小的数字其实很大（或反过来） | 数字即幽默 + 递进王 |
| 规则漏洞 | 消费者不知道的行业潜规则 | 本质剥离 + 定性重锤 |
| 生活冷知识 | 食品、健康、出行中的反常识 | 反常识开场 + 冷幽默 |

### 认知级专用句式库

#### 句式C：反常识开场型

```
公式：[大众认知]。但真实情况是[反常识事实]。
效果：第一句话就制造认知冲突
位置：开场第1-2行

示例：
超市临期牛奶，大家都觉得快过期了。
但真实情况是，保质期6个月的牛奶，
生产后第3个月就算"临期"。它还有3个月才过期。
你不是在买快过期的奶，你是在买打折的新鲜。

有机蔬菜比普通蔬菜贵3倍。
但真实情况是，国内有机认证的抽检率不到5%。
你多花的那两倍钱，买的不是安全，是一个标签。
```

#### 句式D：冷知识递进型

```
公式：你知道[冷知识1]吗？还有更离谱的——[冷知识2]。
效果：层层递进制造"越挖越深"的阅读快感
位置：中段展开

示例：
你知道超市里的"鲜榨果汁"大多是浓缩汁还原的吗？
还有更离谱的——有些"鲜榨"字样旁边写着"风味饮料"，
配料表第一位是水。你花了15块买了一杯糖水。

你知道酒店的"五星级"不是国家统一标准吗？
还有更离谱的——很多"五星"是酒店自己评的，
真正的五星级酒店全国不到800家。你住的可能是个"自称五星"。
```

### 认知级写作公式

```
开场：反常识事实（用句式C，第一句就颠覆认知）
中段：冷知识递进（用句式D，越挖越深）
收尾：追问+价值指引（"下次你去超市，翻过来看配料表"）
```

### 认知级选题清单（常备池）

```
□ 超市里"鲜榨"果汁的真实成本
□ 保质期≠新鲜期，临期食品的真相
□ 你买的"有机"到底有机吗
□ 酒店星级的评定标准漏洞
□ 奶茶店的成本结构（一杯奶茶原料成本不到3块）
□ 化妆品的"专柜价"和"出厂价"差多少
□ 电影票房的分账规则（导演能拿多少）
□ 快递费的真实成本（你以为的8块其实只要2块）
□ 网红餐厅的翻台率才是利润密码
□ 你每天喝的"矿泉水"可能就是自来水
```

## 五、反共识风险红线

### 三类绝对不碰的题材

#### 红线1：国家民生政策、公共福利

```
❌ 为了反转而否定政策导向
   "养老金上涨3%？说白了就是贬值"
   "医保改革是变相涨价"

✅ 可以做：用算账的方式呈现政策对个人的具体影响
   "养老金涨3%，物价涨了多少？我帮你算笔账"
   "医保改革后，门诊报销比例变了，你实际能报多少"
```

#### 红线2：灾难、悲剧、恶性事件

```
❌ 玩冷幽默、找反共识角度
   "XX地震，最可怕的不是地震，是……"
   用本质剥离刀解构悲剧

✅ 可以做：纯事实报道 + 实用信息
   "地震后你家那边的房屋抗震等级是多少？查一下"
```

#### 红线3：特定职业/群体一竿子打翻

```
❌ 身份重定义打击整个群体
   "外卖骑手不是在送餐，是在给算法打工"（对整个群体定性）
   "教师不是在教书，是在完成KPI"

✅ 可以做：对系统/机制提出质疑
   "外卖平台的算法不是在优化配送，是在压缩人的容错空间"
```

### 安全写作三原则

1. **切现象不切人**：靶子是机制/系统/现象，不是具体的个人或群体
2. **算账不定性**：用数据说话，不下道德判断
3. **反共识有度**：反的是"大众认知"，不是"政策导向"和"社会底线"
